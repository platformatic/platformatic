# @platformatic/nitro

Platformatic capability for running [Nitro](https://nitro.build/) applications inside Watt, including Nitro based applications generated by [Lovable](https://lovable.dev/).

Both standalone Nitro applications and Vite applications using Nitro as a Vite plugin are supported.

## Install

```bash
npm install @platformatic/nitro
```
